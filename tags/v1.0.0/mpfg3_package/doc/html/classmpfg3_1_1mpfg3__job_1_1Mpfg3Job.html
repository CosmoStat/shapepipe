Construct a new job for identiofiying a set of files in the dataset to process together.

To obtain the path associated with a given type, just use:

 self.img_path_dico[type]

with one of the aforementioned types.

Parameters:
masterinstance of the Master
datasetdata source used to create the job
img_path_dicofile absolute path dictionary for every file types referenced in the job
argsa list with the branch tree, image number and epoch number
Returns:
an initialized Mpfg3Job object
