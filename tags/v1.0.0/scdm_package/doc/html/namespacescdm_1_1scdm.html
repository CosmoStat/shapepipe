class  MinimizationType
 Define which quantity should be mimimized during fitting and hence the stopping criterion

  • minimize_func_value: minimize the mean variations of the objective function value over a window of the last N function value estimates; the estimate is made against FittingOptions.max_func_tol
  • minimize_param_delta: minimize the mean variation of each parameter over a window of the last N parameter value estimates; the estimate is made against Param.max_tol
  • minimize_param_sigma: minimize the standard deviation of the variations of each parameter over a window of the last N parameter value estimates; the estimate is made against Param.max_tol.
