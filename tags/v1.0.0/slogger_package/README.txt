FileLogger v1.0.0
=================

Author: Marc Gentile (marc.gentile@epfl.ch)

FileLogger is a simple and lightweight file logging facility.

Installation:
-------------

- Download and unpack the file slogger-1.0.0.tar.gz in the directory of your
  choice
- This is a pure Python module and it can then be installed using the standard
  procedure: 

     1. create a console or terminal session
     2. cd to the directory where slogger-1.0.0.tar.gz was detared
     3. enter the command: sudo python setup.py install

  If no root access, use: python setup install --home=<target directory> to 
  install the module to <target directory>. This directory must be included in
  the PYTHONPATH environment variable.
