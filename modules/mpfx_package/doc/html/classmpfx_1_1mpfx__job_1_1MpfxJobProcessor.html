  kwargs

Return the list of MpfxJobResult objects for a specified branch, observation type, data_type and image number.

These parameters must be specified using the "branch=", "img_no=" and "epoch=" keywords

Returns
list of MpfxJobResult objects
1 Examples
2 - self.get_job_results(branch="control/space") to retrieve job results for all images
3  and epochs in branch "control/space"
4 - self.get_job_results(branch="control/space/variable", img_no=1) to retrieve
5  job results of branch "control/space/variable", image number 1, all epochs
6 - self.get_job_results(branch="control/space/variable", img_no=1, epoch=2) to retrieve
7  job results of branch "control/space/variable", image number 1, epoch index 2
Note
MpfxJobResult objects can also be directly obtained from the mpfx.mpfx_job.MpfxJobProcessor.job_result_dico dictionary
