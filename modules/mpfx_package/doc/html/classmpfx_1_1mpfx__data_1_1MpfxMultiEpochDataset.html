Detailed Description

Represent a source dataset that takes care of files with multiple exposures (epochs).

This class is based on the "mono epoch" MpfxDataset class.

def mpfx.mpfx_data.MpfxMultiEpochDataset.query (   self,

Query files matching a set of directory and file patterns, taking care of images with multiple epochs (exposures).

Files with multiple epochs are expected to have format:

1 <image_name>.<image_no>.<epoch_no>.<extension>

where epoch_no represents an epoch index starting with zero.

Parameters
mastermaster object instance
file_search_patternUnix-style file search pattern list (like "*.fits")
dir_list[optional] a list of specific directories to search under base directory
image_lista list of specific image numbers to search, like [0, 100]
image_rangea range of specific numbers images to search like [50, 100]
sort[optional] tell whether to sort the output file names (default True)
recurse[optional] tell whether to walk down directories (default True)
Returns
a list of file paths matching the search criteria
