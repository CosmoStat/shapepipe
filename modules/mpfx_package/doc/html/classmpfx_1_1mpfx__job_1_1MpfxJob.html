def mpfx.mpfx_job.MpfxJob.get_file_components_from_path (   self,
  file_path 
)

Return the list of file components of a file absolute path: file type, image number and possibly epoch index.

1 - If the file is of the form /<path>/<file_type>-<image_no>-<epoch>.<extension> a list
2  [file_type, img_no, epoch] is returned
3 - If the file is of the form /<path>/<file_type>-<image_no>.<extension>, a list
4  [file_type, img_no, None] is returned
Parameters
file_pathabsolute file path of a file (image, catalog, etc.)
1 Examples:
2 - get_file_compoments_from_path("starfield-000.1.fits") => [starfield_image.fits, 0, 1]
3 - get_file_compoments_from_path("star_catalog-000.txt") => [star_catalog.txt, 0, None]
Returns
See also
