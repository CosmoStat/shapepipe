def mpfcs82.mpfcs82_job.MpfCS82JobProcessor.get_job_results (   self,
  kwargs 
)

Return the list of MpfCS82JobResult objects for a specified branch, observation type, data_type and image number.

These parameters must be specified using the "branch=", "img_no=" and "epoch=" keywords

Returns:
list of MpfCS82JobResult objects 
Examples
- self.get_job_results(branch="control/space") to retrieve job results for all images 
  and epochs in branch "control/space"
- self.get_job_results(branch="control/space/variable", img_no=1) to retrieve 
  job results of branch "control/space/variable", image number 1, all epochs 
- self.get_job_results(branch="control/space/variable", img_no=1, epoch=2) to retrieve 
  job results of branch "control/space/variable", image number 1, epoch index 2
Note:
MpfCS82JobResult objects can also be directly obtained from the mpfcs82.mpfcs82_job.MpfCS82JobProcessor.job_result_dico dictionary
