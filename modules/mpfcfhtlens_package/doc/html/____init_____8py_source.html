6 @section Description
7 
8 Mpfx is an extension of the base Mpf multiprocessing framework.
9 
10 
11 @section code Source code
12 
13 Default configuration file:
14 - mpfg3.cfg: configuration file
15 
16 Executable files:
17 - mpfg3_SMP.py: main program for the SMP implementation
18 - mpfg3_MPI.py: main program for the MPI implementation
19 
20 Modules:
21 - mpfg3_args: command-line argument and options
22 - mpfg3_job: parallel job processing
23 - mpfg3_data: @c GREAT3 dataset management
24 
25 The Python program files are located in the ./mpfg3/ directory and the default configuration file
26 @c mpfg3.cfg reside in the ./mpfg3/config directory.
27 
28 @section Prerequisites
29 
30 - Python 2.6 or 2.7 on Unix or Mac OS
31 - mpfg v1.x: base multiprocessing framwework module
32 - sconfig v1.x: configuration file parser module
33 - slogger v1.x: logging module
34 
35 @section Installation
36 
37 - Download and unpack the file mpfg3-2.0.0.tar.gz in the directory of your
38  choice
39 - mpfg3 is a pure Python module and can then be installed using the standard
40  procedure:
41 
42  <ol>
43  <li> create a console or terminal session </li>
44  <li> cd to the directory where mpfg3-2.0.0.tar.gz was detared </li>
45  <li> enter the command: @code sudo python setup.py install @endcode </li>
46  <li> if no root access, use: @code python setup install --home=target_directory
47  @endcode to install the module to target_directory. This directory must be
48  included in the @c PYTHONPATH environment variable </li>
49 </ol>
50 
51 @section Execution
52 
53 The general syntax is:
54 
55  - SMP version: <code>mpfg3_SMP.py [options]</code>
56  - MPI version: <code>mpfg3_MPI.py [options]</code>
57 
58 by default, Mpfx will look for a configuration file named mpfg3.cfg in the
59 ./config directory. The location of this file can be changed using the -c and
60 -d options (see below).
61 
62 The supported options are:
63 
64 <ul>
65  <li> -h, --help Display the usage syntax and the list of supported
66  options. </li>
67 
68  <li> -d, --config-dir Directory where to find the configuration file </li>
69 
70  <li> -c, --config-file Name of the configuration file (default: mpfg3.cfg) </li>
71 </ul>
72 
73 To run mpfg3_MPI.py, one has to use the mpirun executable as:
74 
75  - <code>mpirun -np N mpfg3_MPI.py [options]</code>
76 
77 where N is the number of processors (or nodes) to use. MPI must has been
78 installed and configured appropriately.
79 
80 Since the manager process of Mpfx uses one processor for itself, workers will
81 share N-1 processors.
82 
83 For example: <code>mpirun -np 6 mpfg3_MPI -d mydir -c myconfig.cfg</code> will run mpfg3
84 on 6 processors, 1 for the manager and 5 for the workers. The manager and each
85 of the workers will look for a configuration file: ./mydir/myconfig.cfg.
86 
87 @note Before running Mpfx, edit the configuration file to set the @c BASE_INPUT_DIR value, which
88  should point to the directory where the @c GREAT3 files reside.
89 @note In the configuration file, any environment variable prefixed with @c $ such as @c $HOME will
90  be expanded when read.
